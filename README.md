# Bulk-Ping
A Bash Script to ping single and Multiple Targets from file.

# Usage 

```sh
chmod +x bulk_ping.sh
```
## For single target use as below
```sh
echo target.com | ./bulk-ping.sh
```
## For Bulk targets input from a file
```
./bulk-ping.sh <filename>
```
