# Bulk-Ping
A tool made in bash to ping Multiple Targets from file and output the results in separate files.

# Usage 
```sh
chmod +x bulk_ping.sh
```

## Ping Bulk targets from a file
```
./bulk-ping.sh -f <filename>
```
